# 🔍 Port Scanner Tool

A high-performance **Python-based Port Scanner** that scans target hosts to identify open ports, helping users analyze network exposure and perform basic security assessments. The application features a simple **Tkinter GUI**, multi-threaded scanning for improved speed, and structured scan results for easy analysis.

---

## 🚀 Features

- ⚡ Fast multi-threaded port scanning
- 🌐 Scan any IPv4 address or hostname
- 🔍 Detect open TCP ports
- 🖥️ User-friendly Tkinter GUI
- 📊 Structured scan results
- 🛡️ Useful for network diagnostics and basic security assessments
- ❌ Handles invalid IP addresses and connection errors gracefully

---

## 🛠️ Tech Stack

- **Python**
- **Socket Programming**
- **Tkinter**
- **Multithreading**
- **TCP/IP Networking**

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/Port-Scanner.git
```

### Navigate to the project

```bash
cd Port-Scanner
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python main.py
```

## 📸 Application Workflow

1. Enter the target IP address or hostname.
2. Specify the port range.
3. Start the scan.
4. View detected open ports.
5. Analyze results for network diagnostics.
